# Queries

## Find elements as the user would!

### Types

- getBy - first match, throws error
- queryBy - firts match, returns null
- findBy - returns promise

### Variants

- byLabelText
- byPlaceholderText
- byText
- byAltText
- byTitle
- byDisplayValue (_input, select, textarea value attribute_)
- byRole
